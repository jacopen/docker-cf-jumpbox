# jumpbox in a Docker container for Cloud Foundry

inspired by [jumpbox-boshrelease](https://github.com/cloudfoundry-community/jumpbox-boshrelease).

## to get BASH shell prompt

```
$ docker run -it --rm ozzozz/cf-jumpbox
```
